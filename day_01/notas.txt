1. O que é o Node.js (essência)

Node.js não é uma linguagem e não é um framework.

Ele é um ambiente de execução que permite rodar JavaScript fora do navegador, usando o motor V8 (o mesmo do Chrome).

Características importantes

Assíncrono (non-blocking)

Baseado em eventos

Muito eficiente para APIs e I/O

Exemplo mental

Em vez de “esperar algo terminar”, o Node registra uma tarefa e continua trabalhando.

2. Event Loop (entenda sem trauma)

O Event Loop é o mecanismo que permite ao Node lidar com várias tarefas ao mesmo tempo.

Exemplo simples
console.log('Início');


setTimeout(() => {
  console.log('Timeout');
}, 2000);


console.log('Fim');
Ordem de execução

Início

Fim

Timeout (após 2s)

➡️ O Node não espera o setTimeout terminar.

Mude o tempo para 0

Observe a ordem

3. Assincronismo (fundamental)
3.1 Promise
function delay(ms: number): Promise<string> {
  return new Promise((resolve) => {
    setTimeout(() => resolve('Concluído'), ms);
  });
}
3.2 async / await
async function executar() {
  const resultado = await delay(1000);
  console.log(resultado);
}


executar();
Regra de ouro

Use async/await

Evite callbacks aninhados

4. File System (fs)

O módulo fs permite trabalhar com arquivos.

Ler arquivo
import { readFile } from 'fs/promises';


async function lerArquivo() {
  const conteudo = await readFile('teste.txt', 'utf-8');
  console.log(conteudo);
}


lerArquivo();
Escrever arquivo
import { writeFile } from 'fs/promises';


await writeFile('log.txt', 'Aplicação iniciada');

5. HTTP – base das APIs
O que é HTTP

Comunicação baseada em requisição e resposta.

Métodos mais usados:

GET → buscar

POST → criar

PUT/PATCH → atualizar

DELETE → remover

6. Criando servidor HTTP puro
import http from 'http';


const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ message: 'API funcionando' }));
});


server.listen(3000, () => {
  console.log('Servidor rodando na porta 3000');
});
Teste

Acesse http://localhost:3000

Retorne um objeto com seu nome e data
